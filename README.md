# Bank Statement Downloader

Automates downloading email attachments (bank statements) using Python.

## Features
- Multi-bank support (SVC, Kotak, BOM, Paytm)
- Gmail label-based filtering
- Auto rename (Month + Year)
- Handles PDF & Excel
- Duplicate prevention

## Setup
1. Create `.env`
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py
